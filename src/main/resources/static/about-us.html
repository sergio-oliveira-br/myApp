AluControl

Purpose and Objective

AluControl was developed as part of my academic journey toward completing the Higher Diploma in Science in Computing, specializing in Software Development at the National College of Ireland.

The primary goal of this project is to create a centralized business automation system that optimizes the management of sales and inventory operations, improving efficiency and supporting decision-making through real-time reporting and automation.

This project is not a commercial product but a showcase of my ability to integrate various technologies and frameworks to build a functional system. The main goal here is to demonstrate continuous learning and development through the practical application of academic concepts.

Key Focus Areas

  • Academic Learning Applied in Practice: Each feature of AluControl demonstrates how academic theories are applied in real-world development.
  • Continuous Improvement: The platform is an ongoing project that applies clean code practices, modular design, and user-centric enhancements.

Future Developments: Towards Microservices

Although AluControl is currently a monolithic application, one of the next steps is to evolve towards a microservices architecture. This transition will allow me to explore more advanced design patterns and technologies, such as:

  • Spring Boot, Spring Security, Docker, Spring Cloud, Eureka, PostgreSQL, Maven, JUnit & Mockito, GitHub Actions.

Learn More and Explore

This platform is continuously evolving, with new features and enhancements being added regularly. The goal is not only to build a functional system but to deepen my understanding and practice of software development, security protocols, and system design.

Feel free to explore the application and check out the latest updates on the version history to track the improvements made along the way.
